Статистический анализ исторических данных реального датасета зарядки электромобилей в Англии 
был произведен в среде R с использованием библиотек forecast, lubridate, nortest (carsinpeak.R).

Определение мест зарядных станций было выполнено в Python на примере района Москвы
с использованием библиотеки Plotly 
(https://colab.research.google.com/drive/1I0E87E_rK9TbqCVtqKzpbExjeIOSbdjO?usp=sharing).

Моделирование влияния электромобилей на электрическую сеть, а также методы управления 
зарядными станциями были выполнены в Matlab (метод Монте-Карло, метод бутстрэпа) 
с интеграцией с RastrWin для оценки режима электрической сети (SerenaSK_tarif.m, SerenaSK_V2G.m).
